# TestUtils
A libray of junit extensions.
[![](https://jitpack.io/v/liplum/TestUtils.svg)](https://jitpack.io/#liplum/TestUtils)
## Usage

```groovy
// Step 1
allprojects {
    repositories {
        maven { url 'https://jitpack.io' }
    }
}
// Step 2
dependencies {
    testImplementation 'com.github.liplum:TestUtils:<Tag>'
}
```
